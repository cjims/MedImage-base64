# MedImage-base64

圖片base64編碼

---

## ✨ 功能特色

- 支援多種圖片格式的 Base64 編碼  
- 操作簡單、純本地執行
- 轉碼時不閃退
- 輕量、不需額外安裝大型套件  

---

## 安裝環境
```bash
pip install -r requirements.txt
```

---

### **PNG 轉 ICO**

執行以下指令：

```bash
python png_ico.py
```

---

### **GO EXE**

```bash
python -m PyInstaller --windowed --onefile --icon=pic/app3.ico --name=Base64 main.py
```
